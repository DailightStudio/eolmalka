# 백로그

언젠가 손볼 것들. 끝낸 항목은 **체크 대신 삭제**.

## 데이터 연동 (블로커)

- **환율** — 한국은행 ECOS API 키 발급 → `/api/fx` 엔드포인트. 폴백으로 open exchange rates
- **주유비** — 오피넷 OpenAPI 키 발급 (회원가입+신청 필요) → 전국·시도·시군구 평균
- **금** — 한국금거래소 / KRX 금시장. **공식 API 한정적** → 스크래핑 검토 (robots/약관 확인 선행)
- **항공권** — Skyscanner 파트너 신청 or Amadeus Self-Service or Travelpayouts 어필리에이트. 노선·왕복 최저가 1주일 단위
- 캐싱 — Vercel KV 24h TTL (rate limit 보호)

## 예측 모델

- 1단계: 30일 이동평균 + 계절성 (월별·요일별 평균 비교) — 클라이언트 친화적, LLM 불필요
- 2단계: 시계열 (Prophet 류) — 백엔드 별도 워커 필요할 수 있음
- 항공권은 특수 — "최저가 시즌"이 검색 가능한 정형 정보로 잡혀야 함

## UX

- 카테고리 상세 차트 — Recharts / Apache ECharts 검토
- 가격 알림 등록 (이메일 / 웹푸시)
- 즐겨찾기 카테고리 상단 고정
- 다국어 (en, ja) — 환율·항공권은 외국인 수요도 있음

## 수익화

- 알림 프리미엄 (월정액)
- 항공권 어필리에이트 — 검색→발권 외부 이동 시 커미션
- 광고 (저강도)

## 운영

- 데이터 소스 헬스체크 (env-runtime-setup의 [[server-agent-bot-ops]] 패턴처럼 모니터링 알림)
- API 키 회전 절차 문서화

## 법적·표현

- "투자 자문" 톤 회피. "참고 신호" 정도로 강도 조절
- 데이터 출처·갱신시각·예측 모델 항상 표기
